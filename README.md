# Battleship Game

A classic battleship game with a graphical interface and not so Intelligent AI opponent built with classic JavaScript, HTML & CSS.

## About

This project is a web-based implementation of the classic Battleship game. Play against an not so Intelligent AI opponent that uses targeting algorithm.

Check it out live :https://shriyashzzz.github.io/battleship/

## Technologies Used

- JavaScript (ES6)
- HTML5 & CSS3
- Webpack
- Babel
- Jest (for testing)
- Loadash (for simple array comparison)

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your computer.

### Installation & Running

1. Clone the repository:

```bash
git clone https://github.com/Shriyashzzz/battleship.git
```

2. Navigate to the project folder:

```bash
cd battleship
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm start
```

The game will automatically open in your browser at `http://localhost:8080`

### Other Commands

- **Run tests**: `npm test`
- **Build for production**: `npm run build`

## How to Play

1. Place your ships on your board
2. Click on the enemy's board to attack
3. Sink all enemy ships before they sink yours!
